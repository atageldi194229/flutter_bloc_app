# BLoC pattern in flutter

- App with simple ui
- BLoC pattern used
- Api calls from service [http://jsonplaceholder.typicode.com/](http://jsonplaceholder.typicode.com/)
- Singleton loader
- Error handler
- Data caching using hive
- OOP
- Mockable classes
- Testable blocs
- Simple CI/CD (github actions - Build & Release)
